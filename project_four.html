Shopping cart

Online store shop for indoor plants. You can browse through shop, sort by size of the plant. Add plants to cart and also sign in/out to/from the page.

Project overview

This project was made in react.

I've done the design in figma and experimented with 12 column grid to get the hero section properly divided.

There a few horizontal scroll options. I've added currency - so it gets recognized by where you live. Total amount changes in the cart based by the amount of plants. And you can change quantity in the cart or remove the item.

As soon as you add plants to cart they are stored in local storage. I also implemented the auth0 login. And would like to add storage to a specific user.

This website is also responsive.
