Weather App

Weather app project allows you to check current weather on desired location and also 3 hour and 4 day forecast.

Project overview

This project was the first bigger one where I started working with API's. I used openweatermap api's, more specifically one to get the coordinates when you enter the city name in the input field. Then i take that coordinates and add them to the next api, where i "extract" the weather data from. I use the coordinates in my third api as well, where i get the forecast, 3 hour / 5 day api. I also implemented the geolocation api if the user allows the location to be accessed.

I made the simple and clean design in figma. This project is responsive

    When you search for a city, you can see:
  • current weather and temperature,
  • some basic details like wind speed, pressure, humidity, visibility, min and max temp,
  • todays weather for every three hours,
  • time of sunrise and sunset and,
  • 4 day weather in advance.
