To do list

To do list is an application where you can add all the important to do's so you don't forget.

Project overview

This project definitely took the most time. It has a lot of details and important parts you cannot overlook. The basis of this project were CRUD operations - to be able to create, read, update and delete each project and each task. MVC - to separate view from logic. I used webpack and added date-fns library to easier filter tasks for the coming week.

I made the simple and clean design in figma along with the logo.

This project is the first one that has the local storage, so you can leave tasks on and they will stay there.

I have a lot more ideas for this project, like adding an option to sign in, connecting calendar on the computer/phone and maybe even a quote of the day to make it nicer.

The project is responsive
