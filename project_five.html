Tic tac toe

Tic tac toe is a game where you can play a round against the computer.

Project overview

This game was fun to build, I was introduced to module pattern and factory functions, so i took a bit different approach. The goal was not to have any global variables.

I made my own svgs in figma and added them to css.

I was trying to think outside of the box and make it a bit differently, since everyone uses standard X's and O's, i used owls. You can choose one out of 4 owls.

My next step is to take this game a bit further and implement AI, to make the game more fun.

The whole game is responsive.
