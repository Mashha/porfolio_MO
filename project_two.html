Make your own CV

CV project allows you to make your own CV and print it.

Project overview

CV project was the first project I made with React.

This project was really fun for me. I got an insight into how basic react works and all you can do with it. It allows you to add personal data, previous work experience, education history, skills and an image from the computer.

The design is clean and simple and CV template itself is modern.

I've build this project with react and external file for CSS. Then i changed it to styled components, to learn more and also made one replica with typescript.

This project is not responsive.
